# 2. 서버-클라이언트와 권한 분리
네트워크 게임은 기본적으로 Server-Client 형식을 취한다.
- Server : 고정된 위치에서 서비스를 제공. 
- Client : 서버에 접근해서 서비스를 받음
## 서버의 종류
- Play as Host
	- 호스트가 게임 플레이에 직접적으로 참여
	- 호스트는 로비를 마련함
		- 클라이언트의 요청을 호스트가 받아 각 게임에게 전달함
	- 클라이언트가 속임수를 쓸 수 없는 장점이 있음.
- Dedicated Server
	- 게임 속에 참여하지 않음
	- 네트워크 품질의 향상
