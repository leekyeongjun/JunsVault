# Definition
```asm
STR rt, [rs]
```
- 저장할 값 : 레지스터 rt
- 메모리 주소 : 레지스터 rs
- 저장할 장소 : 메모리상에 레지스터 rs가 값으로 가지고 있는 주소가 가리키는 장소
# Example
```asm
STR r1, [r0]
```

# STR family
| Name | Usage                |
| ---- | -------------------- |
| STR  | store word           |
| STRB | store lower byte     |
| STRH | store lower halfword |
